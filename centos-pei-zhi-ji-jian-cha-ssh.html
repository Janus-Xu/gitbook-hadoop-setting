Centos 配置及检查SSH

Centos 配置及检查SSH

1.检查是否安装 ssh 相关软件包 （openssh-server + openssh-client + openssh）

#yum list installed | grep ssh

[root@VM_224_98_centos ~]# yum list installed | grep ssh
Repodata is over 2 weeks old. Install yum-cron? Or run: yum makecache fast
libssh2.x86_64                    1.4.3-8.el7                         @anaconda 
openssh.x86_64                    6.6.1p1-11.el7                      @anaconda 
openssh-clients.x86_64            6.6.1p1-11.el7                      @anaconda 
openssh-server.x86_64             6.6.1p1-11.el7                      @anaconda

2.检查是否启动 sshd 进程

#ps -ef | grep sshd

3.在 client 端生成公私秘钥对

#ssh-keygen -t rsa -P "" -f ~/.ssh/id_rsa

4.在 ~/.ssh/ 目录下生成 id_rsa（私钥） + id_rsa.pub（公钥）

5.追加公钥到 ~/.ssh/ authorized_keys 文件中

#cat ~/.ssh/id_rsa.pub >> ~/.ssh/authorized_keys

6.修改 authorized_keys 权限为 644

#chmod 644 authorized_keys

7.测试 ssh localhost 免密登录
