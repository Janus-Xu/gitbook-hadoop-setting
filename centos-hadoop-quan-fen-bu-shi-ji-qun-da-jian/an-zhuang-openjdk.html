安装 OpenJDK

安装 OpenJDK 7.0

作用：Hadoop 需要 Java 环境支撑，而 Hadoop2.7.4 最少需要 Java 版本1.8

查看 yum 库中存在那些 JDK 版本

#yum search java|grep jdk

[root@hadoop-master /root]#yum search java|grep jdk
Repodata is over 2 weeks old. Install yum-cron? Or run: yum makecache fast
ldapjdk-javadoc.noarch : Javadoc for ldapjdk
java-1.6.0-openjdk.x86_64 : OpenJDK Runtime Environment
java-1.6.0-openjdk-demo.x86_64 : OpenJDK Demos
java-1.6.0-openjdk-devel.x86_64 : OpenJDK Development Environment
.....

安装 JDK 8.0，默认路径 /usr/lib/jvm/java-1.8.0-openjdk-1.8.0.151-1.b12.el7_4.x86_64

#yum remove java-1.7.0-openjdk
#yum -y install java-1.8.0-openjdk*

查看当前本机可用的 JDK

#alternatives --config java

[root@hadoop-master /usr/lib/jvm]#alternatives --config java

There are 2 programs which provide 'java'.

  Selection    Command
-----------------------------------------------
*+ 1           java-1.8.0-openjdk.x86_64 (/usr/lib/jvm/java-1.8.0-openjdk-1.8.0.151-1.b12.el7_4.x86_64/jre/bin/java)
   2           java-1.8.0-openjdk.x86_64 (/usr/lib/jvm/java-1.8.0-openjdk-1.8.0.151-1.b12.el7_4.x86_64-debug/jre/bin/java)

配置环境变量

1.编辑 /etc/profile 文件

#vim /etc/profile

export JAVA_HOME=/usr/lib/jvm/java-1.8.0-openjdk-1.8.0.151-1.b12.el7_4.x86_64

export PATH=$PATH:$JAVA_HOME/bin

2.使环境变量生效

#source /etc/profile

3.测试是否生效

[root@hadoop-master /usr/lib/jvm]#java -version
openjdk version "1.8.0_151"
OpenJDK Runtime Environment (build 1.8.0_151-b12)
OpenJDK 64-Bit Server VM (build 25.151-b12, mixed mode)
